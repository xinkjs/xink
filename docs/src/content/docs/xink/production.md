## Build

To build the api, run the relevant command per runtime:

- `bun run build`
- `deno task build`
- `npm run build`

## Preview

To preview the api with vite:

- `bun run preview`
- `deno task preview`
- `npm run preview`

To preview the api in your runtime's environment:

- `bun run start`
- `deno task start`
- `wrangler dev`

## Deploy

For Cloudflare, you can deploy your api with `wrangler deploy`.
